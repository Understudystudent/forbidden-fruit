/ GET To display landing page

User Routing

Path HTTP Methods Description
/users/ GET Display all users
/users/:id Fetch Display Single user
/users/register Register Register a User
/users/update/:id Update Update a User
/users/delete/:id Delete Delete an existing User
/users/login Login Login a User

Product Routing

Path HTTP Methods Description
/products/ GET Display all products
/products/:id Fetch Display a single product
/products/addProduct Add Create a new product
/products/update/:id Update Update a product
/products/delete/:id Delete Delete an existing product
