User Routing

Path HTTP Methods Description
/users/ GET To display all users
/users/:id Fetch To display Single user
/users/register Register a User To register a User
/users/update/:id Update Update a User
/users/delete/:id Delete Delete an existing User
/users/login Login To loging in a User

Product Routing

Path HTTP Methods Description
/products/ GET To display all products
/products/:id Fetch To display a single product
/products/addProduct Add To create a new product
/products/update/:id Update Update a product
/products/delete/:id Delete Delete an existing product
